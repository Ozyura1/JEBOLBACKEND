# JEBOLMobile Backend

Backend API untuk aplikasi JEBOLMobile menggunakan Node.js dan Express.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Jalankan server:
   ```
   npm run dev
   ```

Server akan berjalan di http://localhost:5000

## API Endpoints

- GET / : Hello message
- GET /api/users : Get users
